## 🌐 **Release 1.1.3: Thread-Safe Core & Mobile Dashboard**

У цьому оновленні ми залізли глибоко під капот горутин, щоб перемогти дедлоки, вичистити пам'ять від «гонок» і, найголовніше, нарешті відв'язати вас від монітора комп'ютера. Ваш майнер став не тільки стабільнішим, але й мобільнішим!

### ⚙️ **Безпека потоків та Архітектурна стабільність**

* **Race-Safe Memory & Deadlock Elimination:** Повністю переписано логіку роботи з `Mutex` та `RWMutex` в модулях статистики (`stats.go`) та гаманців. Усунено критичну проблему рекурсивних блокувань (Deadlocks), коли `UpdateSingleBalance` утримував м'ютекс під час трансляції оновлень. Тепер транзакції в пам'яті ізольовані, а розсилка івентів відбувається *після* зняття блокувань.
* **Corrupted Memory Fix:** Виправлено фатальну помилку `corrupted double-linked list`, яка виникала через стан гонитви (Data Race) при одночасному записі в мапи декількома горутинами. Вся внутрішня статистика тепер має залізобетонний thread-safe захист.

### 📱 **Мобільний моніторинг (Local Network Access)**

* **Wi-Fi Web Dashboard:** Вбудований HTTP-сервер звільнено від кайданів `localhost`. Тепер він працює на `0.0.0.0`, що дозволяє підключатися до панелі керування майнером з будь-якого пристрою у вашій локальній мережі (наприклад, зі смартфона на дивані).
* **Smart "Copy Link" Button:** Кнопка «Вебдашборд» отримала розумний бекенд-хук. Тепер вона автоматично сканує ваші мережеві інтерфейси, знаходить локальну IPv4-адресу машини (напр. `192.168.x.x`) та миттєво копіює готове посилання для мобільного браузера прямо в буфер обміну.

### 🎨 **UI/UX та Linux Rendering**

* **Anti-Banding & GPU Optimization:** Вирішено проблему візуальної «зебри» (сходинок на градієнтах) за допомогою мікро-шуму та CSS-дизерування. Додатково стабілізовано рендеринг у середовищі Linux (WebKitGTK/Wayland), щоб інтерфейс використовував апаратне прискорення GPU без мікро-фризів.

📦 **Вміст релізу**

- `SHMiner-windows-amd64-setup.exe` — **Рекомендовано!** Повноцінний інсталятор (створює ярлики, зберігає налаштування).
- `SHMiner-windows-amd64-portable.exe` — Портативна версія (працює прямо з папки без встановлення).
- `SHMiner-windows-arm64-setup.exe` — Повноцінний інсталятор під Windows ARM.
- `SHMiner-windows-arm64-portable.exe` — Портативна версія для Windows ARM.
- `SHMiner-linux-amd64` — Версія для Linux x64 (портативна. Автоматичне встановлення на Linux ще не реалізовано).
