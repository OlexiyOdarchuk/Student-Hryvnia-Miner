# S-UAH Miner — First Public Release

![App Icon](build/appicon.png)

🚀 **S-UAH Miner** — це високопродуктивний клієнт для майнінгу студентської внутрішньої валюти, який поєднує швидкість, стабільну обробку та мінімальне навантаження на систему. Проєкт створено як технічний експеримент і демонструє, як Go + Wails + Svelte можуть служити основою для реального десктопного майнера.

## 🔥 Що це
S-UAH Miner концентрується на зовнішньому вигляді, керованих гаманцях і надійній взаємодії з мережевими вузлами. Він підходить для локального запуску, демонструє живий дашборд і не залежить від сторонніх серверів — усе зберігається в локальному зашифрованому сховищі.

## ⚙️ Основні можливості
*   ⚡ **Максимально швидкий майнінг** — оптимізовані обчислення SHA-256 для багатоядерних CPU.
*   🧵 **Паралельна обробка** — worker pool на goroutines і атомарні метрики забезпечують стабільний хешрейт.
*   💼 **Розширене управління гаманцями**:
    *   Створення та імпорт гаманців (JSON або вручну).
    *   Експорт ключів та даних для перенесення.
    *   Перемикання, перейменування та синхронізація кількох гаманців.
*   📊 **Веб-дашборд** — живі метрики, баланси та історія хешрейту через вбудований HTTP/WS сервер.
*   🔄 **Автоматичне оновлення конфігурації** — параметри майнінгу і частота оновлення балансу перезапускаються без перезавантаження.
*   🧠 **Розумна обробка помилок і повторів** — backoff, повторні спроби та локальні логи.
*   🪟 **Підтримка Windows / Linux** (Wails) — один і той самий код компілюється для всіх платформ.
*   🔐 **Зашифроване збереження** — усі гаманці й налаштування шифруються перед записом у `SHMinerSettings.bin`.

## 🧪 Технічні деталі
*   **Backend**: Go (1.25.6) з багатопоточними службами (`wallets`, `stats`, `miner`, `nodeclient`).
*   **UI**: Wails + Svelte + Vite з TypeScript.
*   **Hashing**: SHA-256 (Go `crypto/sha256`).
*   **Concurrency**: goroutines + atomic/Mutex + worker pool.

## ⚠️ Важливо
*   Проєкт має демонстраційну цінність (~освітні/дослідні цілі).
*   Автор не несе відповідальності за стороннє використання або діяльність, що порушує правила сервісів.
*   Конфіденційні дані зберігаються тільки локально, у зашифрованому файлі, і не передаються третім особам.

## 🛠️ Встановлення та збірка

### Попередні вимоги
*   Go 1.25.6+
*   Node.js 20+
*   Wails CLI (`go install github.com/wailsapp/wails/v2/cmd/wails@latest`)

### Розробка
```bash
cd frontend
npm ci
cd ..
wails dev
```

### Продакшн-збірка
```bash
cd frontend && npm run build
cd ..
wails build -ldflags "-s -w"
```
Готовий бінар буде у `build/bin`.

## 🎮 Використання
1.  Запустіть майнер (`wails dev` у режимі розробки або зібраний `.exe`/`SHMiner`).
2.  Оберіть пароль сесії (ключ до шифрування `SHMinerSettings.bin`).
3.  Створіть або імпортуйте гаманці (вкладка "Гаманці").
4.  Переключіть гаманці, активуйте майнінг і спостерігайте метрики на Dashboard/Statistics.
5.  Логи доступні у вкладці Logs або через API `log`/`stats` події Wails.
